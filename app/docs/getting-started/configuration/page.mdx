# Configuration

## MemoryStore options

```python
store = MemoryStore(
    database_url="postgresql://user:pass@localhost/db",

    # Embedding model (default: all-MiniLM-L6-v2, 384 dims, local ONNX)
    embedding_model="all-MiniLM-L6-v2",

    # Cross-encoder reranker (adds ~10ms but improves precision)
    reranker_enabled=True,
    reranker_model="cross-encoder/ms-marco-MiniLM-L-6-v2",

    # Connection pool
    pool_min_size=1,
    pool_max_size=10,

    # Retrieval tuning
    ef_search=100,                # HNSW recall parameter
    rrf_k=60,                     # Reciprocal Rank Fusion k

    # Channel weights for hybrid retrieval
    channel_weights={
        "semantic": 1.0,
        "bm25": 0.8,
        "entity": 0.6,
        "temporal": 0.4,
    },

    # Type boosts applied post-fusion
    type_boosts={
        "insight": 1.5,
        "event": 1.0,
        "raw": 0.5,
    },

    # Quotas
    max_writes_per_minute=100,
    max_memories_per_agent=10_000,

    # Recall cache
    recall_cache_ttl=60.0,
    recall_cache_size=1000,
)
```

## Custom embedder

```python
from unforget import MemoryStore, OpenAIEmbedder

store = MemoryStore(
    "postgresql://...",
    embedder=OpenAIEmbedder(model="text-embedding-3-large"),  # 3072 dims
)
```

Or implement `BaseEmbedder` for any provider:

```python
from unforget.embedder import BaseEmbedder

class MyEmbedder(BaseEmbedder):
    def embed(self, text: str) -> list[float]:
        ...

    def embed_batch(self, texts: list[str]) -> list[list[float]]:
        ...

    @property
    def dims(self) -> int:
        return 768
```

## Consolidation scheduler

```python
from unforget import ConsolidationScheduler

scheduler = ConsolidationScheduler(
    store,
    interval_seconds=3600,       # run every hour
    write_threshold=50,           # or after 50 writes
    llm=my_llm_callable,          # optional: enables promotion
    similarity_threshold=0.92,    # dedup cosine threshold
)
store.attach_scheduler(scheduler)
await scheduler.start()
```
